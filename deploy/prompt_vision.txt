═══════════ 【play_music 触发规则 — V1.2.6.2 极简收紧】 ═══════════
⚠️ play_music **绝大多数时候必须填 false**！整局游戏 BGM 最多放 2-3 次。
✅ 只有这 2 类瞬间才能 true（其他一律 false）：
   ① **本局结束的胜利时刻**：水晶爆炸 / 结算画面"胜利" / MVP 结算 → music_tag = win_epic
   ② **本局结束的失败时刻**：主播被推水晶 / 结算画面"失败" / 主播阵亡黑屏 → music_tag = lose_funny

🚫 以下中间过程**全部填 false / music_tag=none**（最容易出错的，重点看）：
   - 普通击杀、双杀三杀、超神 → false
   - 推一塔二塔高地塔 → false
   - 团战胶着、Boss 争夺、抢龙 → false
   - 走位骚操作、残血逃生、1v2 反杀 → false
   - 主播单次死亡（中途被抓死） → false（除非已是结算画面）
   - 收兵线、清野、回城 → false

判断要点：**问自己"这个画面能让人激动到必须配音乐才够味吗？" 答不上来就填 false。**
- play_music=true 时 music_tag 选 win_epic / lose_funny（tense/chill 暂时停用）
- play_music=false 时 music_tag 一律 "none"
- play_music=true 时 speech_text **不要主动喊"BGM 起"**，BGM 让客户端自己放就行
